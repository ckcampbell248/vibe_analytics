# Retail Vibe Analytics

Demo scripts showing how to use AI-assisted analytics with the [Olist Brazilian E-Commerce dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) from Kaggle.

## What's Inside

| File | Description |
|------|-------------|
| [Retail Vibe Analytics.md](Retail%20Vibe%20Analytics.md) | Original demo script designed for Microsoft Fabric Lakehouse + Spark |
| [Retail Vibe Analytics - Local.md](Retail%20Vibe%20Analytics%20-%20Local.md) | Local notebook edition — uses CSV files + pandas, no cloud dependencies |

## Quick Start (Local Edition)

1. Download the dataset from [Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) and unzip it
2. Install dependencies: `pip install pandas matplotlib seaborn plotly scikit-learn statsmodels`
3. Open the folder in VS Code with the Python and Jupyter extensions installed
4. Create a new Jupyter notebook and start following the prompts in the script

See [Retail Vibe Analytics - Local.md](Retail%20Vibe%20Analytics%20-%20Local.md) for full setup instructions and all demo prompts.
